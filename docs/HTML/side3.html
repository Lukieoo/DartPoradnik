4.0 Instrukcje warunkowe

Tak samo co w języku Java ,C++

Zanim przejdę do opisania instrukcji warunkowych przeczytasz coś więcej na temat operatorów logicznych.

  • == równe,
  • < mniejsze,
  • <= mniejsze bądź równe,
  • › większe,
  • ›= większe bądź równe,

Proste operacje logiczne możesz ze sobą łączyć przy pomocy dodatkowych operatorów:

  • && i
  • || lub,

4.1 IF,ELSE

void main() {
  int a=0;
  if(a==0){
    //Jeżeli jest równe 0 wykona to
  }else if(a==1){
    //Jeżeli jest równe 1 wykona to
  }else{
    //W każdym innym przypadku wykona to
  }
}
Tak samo jak w Javie czy też c++
